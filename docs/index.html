Mercapp - Micrositio del Proyecto

Resumen técnico

Mercapp es un sistema compuesto por:

  • Frontend: Vue 3 + Vite (desplegado en Netlify)
  • Backend: Node.js + Express (desplegado en Railway)
  • Base de datos: MongoDB Atlas

Guía de ejecución local

# Clonar el repositorio
git clone https://github.com/usuario/mercapp.git

# Frontend
cd mercapp-frontend
npm install
npm run dev

# Backend
cd mercapp-backend
npm install
npm run dev

Frontend en Netlify: mercapp.netlify.app

API base en Railway: mercapp-api.up.railway.app
